# Vibe Coding 100 - 小项目挑战计划

> 100 个创意小项目，探索编程的无限可能

## 📖 项目简介

这是一个 **100 个小项目** 的编程挑战计划。每个项目都是独立、完整、有趣的小型应用，涵盖游戏开发、工具实用、创意艺术等多个领域。

**项目理念**：
- ✨ **小而美**：每个项目专注于一个核心功能，做到极致
- 🎯 **可运行**：所有项目都是完整可运行的，不是半成品或 demo
- 🚀 **纯前端**：使用 HTML/CSS/JavaScript 等前端技术，无需后端部署
- 📚 **可学习**：代码清晰、结构良好，适合学习和参考

## 🎮 项目列表

### 1. 华夏争霸：地缘战略 (Huaxia Hegemony)

一个三国题材的实时战略模拟游戏，通过智能 AI 系统模拟多势力争霸。

**技术亮点**：
- Canvas 2D 程序化地图生成（Voronoi 算法）
- 四种 AI 决策策略（征伐/固守/拓荒/休养）
- 拦截作战机制
- 粒子系统实现军队移动和战斗效果

**运行方式**：直接打开 `war-simulation/index.html`

**在线体验**：[打开游戏](https://winston-wwzhen.github.io/vibe_coding_100/war-simulation/index.html)

---

### 2. 手势翻页漫画书 (Gesture Comic Reader)

一个支持触摸手势和鼠标拖拽的交互式漫画阅读器，展示78张精美扑克牌。

**技术亮点**：
- 流畅的手势识别算法（滑动距离+速度检测）
- 3D 卡片翻页动画（CSS3 Transform）
- 实时拖拽反馈和物理效果
- 智能图片预加载策略
- 完整的触摸和鼠标事件支持

**运行方式**：直接打开 `漫画书/index.html`

**操作方式**：
- 🖱️ 桌面端：鼠标拖拽、方向键、点击按钮
- 📱 移动端：左右滑动、点击按钮
