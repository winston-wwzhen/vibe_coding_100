# 华夏争霸：地缘战略

一个基于 HTML5 Canvas 的三国题材实时战略模拟游戏。

## 游戏特色

- **真实战争模拟**：多势力争霸，每方势力根据自身情况制定战略决策
- **智能 AI 系统**：四种策略模式（征伐/固守/拓荒/休养），动态调整战术
- **拦截作战机制**：当敌军经过友方领土时，州府会主动派兵迎击
- **丰富地形系统**：荒野、粮仓、关隘、商都四种地形，各有特色
- **动态叛乱**：大帝国概率分裂，死掉的势力可能重新崛起
- **随机事件**：屯田、旱灾、瘟疫、求贤等历史事件影响战局

## 游戏玩法

### 操作说明

- **左键点击**省份：天赐神兵（增加500兵力）
- **右键点击**省份：天赐神兵（增加500兵力）
- **Shift + 右键**：天降正义（减少500兵力）
- **鼠标悬停**：查看省份详细信息

### 速度控制

- `1x` - 慢速
- `4x` - 正常
- `MAX` - 最快
- `⏸` - 暂停

### AI 决策策略

- **⚔️ 征伐** - 兵力充足时主动进攻敌对势力
- **🛡️ 固守** - 领土较少或首都受威胁时采取防御态势
- **🌾 拓荒** - 优先占领富庶的粮仓和商都
- **💤 休养** - 兵力不足时恢复发展，积蓄力量

### 地形类型

| 地形 | 图标 | 特点 |
|------|------|------|

| 关隘 | 🛡️ | 防御 2.5x，易守难攻 |

### 势力介绍

- **蜀汉** - 红色，西南崛起

## 游戏规则

1. 每个势力自动发展兵力，扩张领土
2. 省份兵力超过上限时，会自动向邻近地区派兵
3. 占领敌对省份需要击败其驻军
4. 首都会随领土中心动态迁移
5. 失去所有领土的势力灭亡
6. 最后存活的势力实现天下统一

## 技术实现

- **纯前端**：单 HTML 文件，无需后端
- **Canvas 渲染**：1024×768 分辨率，200 个省份
- **程序化地图**：使用噪声算法生成中国地图形状
- **粒子系统**：实时渲染军队移动和战斗

## 运行方式

直接用浏览器打开 `华夏争霸.html` 即可游玩。

支持所有现代浏览器（Chrome、Firefox、Edge、Safari）。

## 开发说明

本项目采用原生 JavaScript 开发，无外部依赖。

主要技术点：
- Canvas 2D 绘图
- Voronoi 图算法生成省份边界
- A* 寻路算法（简化版）
- 自定义粒子系统
- 模糊逻辑 AI 决策
